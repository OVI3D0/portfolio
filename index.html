Pomodoro Website

A website inspired by one I used myself to study. The website has a 25 minute timer which counts down from 25 minutes to 0, playing a chime at the end to let the user know that the timer has finished counting. The site also plays background videos to set a calming, study mood for the user, which they can also change at any time.

Roofing Website

A pinterest-inspired mock landing page for a roofing company based in Connecticut. The website features several sections, advertising the various services this company offers, as well as ways to contact the company (Phone numbers used are fake).

Tindog Website

A Tinder-inspired website made when I was following a web development course. The site uses HTML and CSS to make a mock landing page for a tinder-like website, but for dogs.
